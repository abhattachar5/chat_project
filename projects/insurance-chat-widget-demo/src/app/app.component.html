Insurance Chat Widget - Phase 1 Demo

This demo showcases the Phase 1 foundation: Widget Shell Component with Floating Action Button and Expandable Panel

Features Implemented in Phase 1

  • ✅ Floating Action Button (FAB) - Minimized state
  • ✅ Expandable Chat Panel - Expanded state
  • ✅ Material Toolbar with header
  • ✅ Minimize and Close controls
  • ✅ Responsive design (mobile/desktop)
  • ✅ Theme system with CSS custom properties

How to Use

  1. Look at the bottom-right corner for the Floating Action Button (FAB)
  2. Click the FAB to expand the chat panel
  3. Use the minimize button (─) to minimize the panel
  4. Use the close button (×) to close the widget
